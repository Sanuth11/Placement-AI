# 🚀 Placement AI — AI Career Copilot

Placement AI is an **AI-powered career assistant** designed to help students and job seekers improve their chances of landing jobs.
It provides resume analysis, interview preparation, job recommendations, and more — all in one platform.

---

# 🌟 Features

### 📄 Resume Analyzer

* Upload resume (PDF)
* AI-powered analysis
* ATS Score
* Strengths & Weaknesses
* Improvement Suggestions

### ✨ Resume Optimizer

* Upload Resume + Job Description
* AI suggests improvements
* Tailored resume recommendations

### 🎤 Mock Interview

* AI-generated interview questions
* Based on skills & role
* Realistic interview simulation

### 📊 Answer Evaluation

* Submit your answers
* AI evaluates responses
* Provides feedback & score

### 💼 Job Recommendations

* AI suggests job roles
* Based on resume & skills

### 🤖 AI Career Chat

* Ask career-related questions
* Get AI guidance

### 🕘 History

* Resume analysis history
* User-specific data

### 🔐 Authentication

* Signup/Login
* Secure JWT Authentication
* Multi-user support

# ⚙️ Installation

## Clone Repository

```
git clone https://github.com/your-username/placement-ai.git
```

---

# Backend Setup

```
cd backend
npm install
```

Create `.env`

```
MONGO_URI=your_mongodb_connection
GROQ_API_KEY=your_groq_api_key
```

Run backend

```
npm start
```

---

# Frontend Setup

```
cd frontend
npm install
npm start
```

---

# 🔑 Environment Variables

Create `.env` in backend

```
MONGO_URI=
GROQ_API_KEY=
JWT_SECRET=
```
